# User Contribution Guidelines

Welcome to the **Image Transformer & Animator** project! We appreciate your interest in contributing.

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template.
- Provide a clear description and steps to reproduce.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Explain the value of the enhancement.

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/your-feature`).
3. Commit your changes (`git commit -m 'Add some feature'`).
4. Push to the branch (`git push origin feature/your-feature`).
5. Open a Pull Request.

## Development Setup

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
4. Build for production: `npm run build`.

## Code Style
- Follow the existing React/TypeScript patterns.
- Ensure all types are properly defined.
- Run `npm run build` to verify no errors before submitting.
